# Todo List

https://vinivin153.github.io/todolist-react/

### 구현 기능
* 추가
* 수정
* 삭제

### 구현 방법
* styled-components를 이용한 CSS-in-JS
* React.memo와 useState의 Functional updates를 이용한 컴포넌트 최적화(리렌더링 줄이기)
* 불변성을 지키기 위한 spread operator사용

### 사용한 ES6
* Variables const
* Arrow function
* Template literal
* Destructuring assignment
* Spread operator
